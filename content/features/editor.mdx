# Rich Editor

Reflecto's editor is built on [TipTap](https://tiptap.dev) — a headless, extensible rich text framework. It provides a premium writing experience with markdown shortcuts, inline formatting, and smart extraction.

## Writing Basics

The editor supports standard markdown shortcuts that convert as you type:

| Shortcut | Result |
|----------|--------|
| `# ` | Heading 1 |
| `## ` | Heading 2 |
| `### ` | Heading 3 |
| `**text**` | **Bold** |
| `*text*` | *Italic* |
| `~~text~~` | ~~Strikethrough~~ |
| `- ` or `* ` | Bullet list |
| `1. ` | Numbered list |
| `` `code` `` | Inline code |
| `> ` | Blockquote |
| `---` | Horizontal rule |

## Formatting Toolbar

The editor includes a floating toolbar with buttons for:
- Bold, Italic, Underline, Strikethrough
- Headings (H1, H2, H3)
- Bullet list, Ordered list
- Blockquote
- Code block
- Link insertion

## @Mentions — People

Type `@` followed by a name to mention someone. The editor shows a **suggestion dropdown** with existing people from your database:

- Selecting an existing person links to their profile
- Typing a new name creates a new Person record automatically
- People are extracted and stored as relationships on the entry
- Mentioned people appear in the **People** management page

<Callout type="info">
People mentioned with `@` are extracted and tracked across all your entries. Visit the People page to see everyone you've mentioned and how often.
</Callout>

## #Tags — Auto-Extraction

Use `#tagname` anywhere in your content to tag an entry:

- Tags are **automatically extracted** from content when you save
- Existing tags appear in a suggestion dropdown
- New tags are created on-the-fly
- Each tag maintains a count of entries it's associated with
- Tags are scoped per user — your tags are private

## Image Uploads

Reflecto supports inline images with these upload methods:

- **Drag & Drop** — Drag an image file directly into the editor
- **Paste** — Paste an image from your clipboard

### How It Works

1. When you add an image, it's uploaded to **ImageKit** (cloud image CDN)
2. A thumbnail is automatically generated
3. The image appears inline in your content with a resizable container
4. Images are stored as `Attachment` records linked to your entry

<Callout type="info">
Supported file types include common image formats (JPEG, PNG, GIF, WebP). Files are stored securely in your ImageKit account.
</Callout>

## Bullet Counting

When **Bulleted Mode** is enabled in Settings, the editor tracks and displays the number of bullet points in your entry. This appears as a count (e.g., `#15`) in the header, encouraging consistent, structured writing.

## Auto-Save

The editor uses **debounced auto-save** — after you stop typing for a brief moment, your content is automatically saved to the server via a tRPC mutation. The cloud sync indicator in the header shows real-time save status.

## Link Embeds

Paste a URL and the editor will create a clickable link. The Link extension supports:
- Click-to-open links
- Link editing and removal
- Auto-detection of URLs
