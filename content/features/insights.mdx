# Insights & Analytics

The Insights dashboard transforms your entries into visual analytics, helping you discover patterns in your reflective practice.

## Dashboard Sections

The Insights page has three main sections, stacked vertically:

### 1. Activity Heatmap

A GitHub-style **contribution heatmap** showing your writing activity over time:

| Level | Color Intensity | Meaning |
|-------|----------------|---------|
| 0 | Empty / Muted | No entries |
| 1 | 20% primary | Low activity (1 entry) |
| 2 | 40% primary | Moderate activity |
| 3 | 60% primary | High activity |
| 4 | 100% primary | Peak activity |

The heatmap always shows **all entries** regardless of the active tab filter, acting as an overall activity tracker.

### 2. Stats Grid

Key metrics calculated from your filtered entries:
- Total entries count
- Average entries per day/week
- Most active day
- Streak information

### 3. Insight Charts

Visual charts showing trends and distributions across your entries, including writing frequency over time and entry type breakdowns.

## Filtering by Type

Use the **tab bar** at the top of the Insights page to filter analytics by entry type:

| Tab | Filters To | Color |
|-----|-----------|-------|
| **Journal** | Journal entries | Default |
| **Dreams** | Dream entries | Yellow |
| **Highlights** | Highlight entries | Red |
| **Tags** | All entries (tag analysis) | Blue |
| **People** | All entries (people analysis) | Emerald |
| **Wisdom** | Wisdom entries | Pink |

<Callout type="info">
The Tags and People tabs show all entries but focus their analysis on tag/people frequency and distribution rather than filtering by entry type.
</Callout>

## The Reflect Page

Separate from the Insights dashboard, the **Reflect** page provides a **timeline view** of your recent entries across all types.

### How It Works

- Fetches entries from the current week (Monday to today)
- Groups entries chronologically
- Shows entry type icons (Journal, Dream, Highlight, Idea, Wisdom)
- Each entry links to its full view in the Write page
- Includes filter options and a summary header

### Entry Type Icons in Timeline

| Type | Icon |
|------|------|
| Journal | BookOpen |
| Dream | Moon |
| Highlight | Heart |
| Idea | Lightbulb |
| Wisdom | Gem |

## Activity Tracking System

Behind the scenes, Reflecto maintains detailed activity data:

### Activity Logs
- One record per user per day
- Tracks `entryCount` — total entries that day
- Tracks `entryTypes` — breakdown by type (e.g., `{ journal: 3, dream: 1 }`)
- Feeds the heatmap visualization

### Streaks
- Tracks consecutive days of writing
- Each streak has a `startDate`, `endDate`, and `length`
- An `isActive` flag marks the current streak
- Displayed on the Journal page via the Streak Bar component
