# Workspaces

Reflecto organizes your writing into **six distinct workspaces**, each designed for a different type of reflection. All workspaces share the same powerful rich editor but have unique metadata and purpose.

## Overview

| Workspace | Purpose | Unique Metadata |
|-----------|---------|-----------------||
| **Journal** | Daily reflections | Category, mood (1-5), bullet count |
| **Dreams** | Future aspirations | Bullet count |
| **Highlights** | Meaningful moments | Importance level, linked entry |
| **Ideas** | Creative brainstorms | Status tracking |
| **Wisdom** | Collected knowledge | Type, author, source |
| **Notes** | Quick captures | Pin status, color |

---

## Dreams

The Dreams workspace is for **future aspirations** — your big-picture visions, goals, and ambitions.

Each dream entry is a rich-text document you can revisit and expand over time. Access your dreams list at `/dreams`, where entries are shown with:
- Title and content preview
- Creation date
- Star status
- Delete option

Create a new dream by clicking the **+** button, which opens the Write page with `?type=dream`.

---

## Highlights

Capture **meaningful moments** — an inspiring conversation, a breakthrough, a special experience.

### Importance Levels

Each highlight can be tagged with an importance level:

| Level | Use For |
|-------|---------|
| **High** | Life-changing or deeply significant moments |
| **Medium** | Notable events worth remembering |
| **Low** | Small but meaningful occurrences |

### Linked Entries

Highlights can be linked to existing journal entries via the `linked_entry_id` metadata field, connecting a moment to the day it happened.

---

## Ideas

Track creative sparks from inception to development.

### Status Tracking

Every idea has a **status** that tracks its progression:

| Status | Description |
|--------|-------------|
| **Contemplating** | Just thinking about it |
| **Seed** | An early-stage idea worth nurturing |
| **Reflecting** | Actively developing the concept |
| **Draft** | Ready to take action |

You can also use custom status values — the field accepts free-form text.

The Ideas page displays entries sorted by date with visible status badges and star indicators.

---

## Wisdom

A curated collection of knowledge — things you've learned, read, or observed.

### Wisdom Types

Each wisdom entry has a specific type with distinct visual styling:

| Type | Purpose |
|------|---------|
| **Thought** | Your own philosophical musings |
| **Quote** | Words from others that resonate |
| **Fact** | Interesting factual knowledge |
| **Excerpt** | Passages from books or articles |
| **Lesson** | Life lessons learned from experience |

### Additional Metadata

- **Author** — Who said or wrote it (for quotes and excerpts)
- **Source** — Where you found it (book, article, conversation)

Create wisdom entries directly from the Wisdom page using the action buttons for each type.

---

## Notes

Quick, lightweight entries for capturing information that doesn't fit other categories.

### Special Features

- **Pinning** — Pin important notes to keep them at the top of your list
- **Colors** — Assign a color to visually distinguish notes
- **Sorting** — Sort notes by date (newest/oldest) with the sort toggle

<Callout type="info">
Both Dreams and Notes can be toggled on/off in **Settings** if you don't need them. When disabled, they're hidden from the sidebar.
</Callout>

---

## Shared Features Across All Workspaces

Every workspace shares these capabilities:

- **Rich Text Editor** — Full TipTap editor with markdown, formatting, and media
- **Starring** — Mark any entry as important
- **Tags** — Use `#tag` in your content to auto-extract tags
- **People** — Use `@name` to mention and track people
- **Image Uploads** — Attach images via drag-and-drop or paste
- **Soft Delete** — Entries are soft-deleted (recoverable) before permanent removal
- **Auto-Save** — Content syncs automatically via debounced mutations
