# Your First Entry

Learn how to create your first reflection in Reflecto. This guide covers the Write page, editor basics, and entry types.

## The Write Page

After signing in, you'll land on the **Write** page — your primary workspace for capturing thoughts and reflections.

<Callout type="info">
The Write page is designed to be distraction-free. The editor takes center stage with minimal UI elements.
</Callout>

### Interface Overview

Here's what you'll see on the Write page:

```
┌─────────────────────────────────────────────┐
│  Fri, Mar 7              ☁️ Saved    ⭐     │
│  ──────────────────────────────────────────  │
│  Friday                                      │
│                                              │
│  [ Journal ▼ ]  [ Category ▼ ]  [ Mood ]   │
│                                              │
│  ┌──────────────────────────────────────┐   │
│  │  Start writing...                    │   │
│  │                                       │   │
│  │  The editor supports markdown,       │   │
│  │  **bold**, *italic*, lists, and more │   │

│  └──────────────────────────────────────┘   │

└─────────────────────────────────────────────┘
```

### Header Elements

| Element | Description |
|---------|-------------|
| **Date Display** | Current date in small mono font, day in large serif italic |
| **Special Date Badge** | Pulsing ✨ sparkle on holidays (if enabled in settings) |
| **Cloud Sync** | Green checkmark when your entry is saved |
| **Star Icon** | Click to mark this entry as important |
| **Bullet Count** | Shows `#42` when bulleted mode is active |

## Creating Your First Entry

<Steps>
### Start Writing

Click in the editor area and start typing. The editor automatically focuses when you load the page.

```markdown
Today I started using Reflecto. I'm excited to build a journaling habit!

This editor supports **bold**, *italic*, and even lists:
- Privacy-first design
- Beautiful interface
- Connected thinking
```

The editor will render your formatting in real-time.

To change the entry type, use the dropdown at the top of the editor.

### Add Tags and People

As you write, you can reference tags and people using special syntax:

- **Tags** — Type `#` followed by the tag name (e.g., `#work`, `#fitness`, `#learning`)
- **People** — Type `@` followed by the person's name (e.g., `@Alice`, `@Bob`)

```markdown
Had a great meeting with @Sarah about the #project. 
Feeling grateful for #teamwork.
```

<Callout type="info">
Tags and people are automatically extracted from your entries and can be managed from the Tags and People pages in the sidebar.
</Callout>

### Set Metadata (Optional)

Depending on the entry type, you can set additional metadata:

**For Journal entries:**
- **Category** — Thoughts, Gratitude, Reflections, Daily Log, etc.
- **Mood** — Rate your mood from 1-5

**For other entry types:**
- **Importance** (Highlights) — Low, Medium, High
- **Status** (Ideas) — Seed, Draft, Refined, Implemented
- **Wisdom Type** (Wisdom) — Quote, Thought, Fact, Excerpt, Lesson
- **Color** (Notes) — Choose from multiple colors
- **Pin** (Notes) — Pin important notes to the top

### Auto-Save

Reflecto automatically saves your entry as you type. Look for the cloud sync indicator in the header:

- **Saving...** — Entry is being synced
- **✓ Saved** — Entry is safely stored

<Callout type="success">
No need to click a save button! Your work is continuously backed up.
</Callout>

### Star Important Entries

Click the star icon (⭐) in the header to mark this entry as important. Starred entries can be filtered across all views.

</Steps>

## Editor Basics

The Reflecto editor is powered by TipTap and supports rich text formatting:

### Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl/Cmd + B` | **Bold** text |
| `Ctrl/Cmd + I` | *Italic* text |
| `Ctrl/Cmd + U` | <u>Underline</u> text |
| `Ctrl/Cmd + K` | Insert link |
| `Ctrl/Cmd + Z` | Undo |
| `Ctrl/Cmd + Y` | Redo |
| `Ctrl/Cmd + Shift + 8` | Bullet list |
| `Ctrl/Cmd + Shift + 7` | Numbered list |

### Markdown Support

You can also use markdown syntax:

```markdown
# Heading 1
## Heading 2
### Heading 3

**bold** or __bold__
*italic* or _italic_

- Bullet list item
- Another item

1. Numbered list
2. Next item

[Link text](https://example.com)

> Blockquote
```

### Image Uploads

To add images to your entry:

1. Click the image icon in the toolbar (or drag & drop)
2. Select an image file from your device
3. The image will be uploaded to ImageKit CDN
4. It appears inline in your entry

<Callout type="warning">
Image uploads require an active internet connection. The first upload may take a few seconds.
</Callout>

### Bulleted Mode

For quick, list-style entries, enable **Bulleted Mode**:

1. Click the bullet icon in the header
2. Each paragraph becomes a bullet point
3. A bullet count (`#42`) appears in the header

This is perfect for daily logs and quick captures.

## Viewing Your Entry

After creating your entry, you can view it in different ways:

### From the Journal Page

1. Click **Journal** in the sidebar
2. Navigate to today's date
3. Your entry appears in the list

### From the Reflect Page

1. Click **Reflect** in the sidebar
2. See your entry in the timeline view

### From Entry Type Pages

1. Navigate to the entry type page (e.g., Dreams, Highlights)
2. Your entry appears in the corresponding workspace

## Editing Existing Entries

To edit an entry you've already created:

1. Navigate to the entry from Journal, Reflect, or a workspace page
2. Click the entry to open it
3. Make your changes in the editor
4. Changes are auto-saved

<Callout type="info">
When editing an existing entry, you'll see a **Linked Context** section below the editor showing related entries from the same date.
</Callout>

## Tips for Better Entries

### Be Consistent
Try to write every day, even if it's just a few sentences. Consistency builds the habit and helps Reflecto track your streaks and patterns.

### Use Tags Liberally
Tags help you connect thoughts across entries. Use them for topics, projects, emotions, or anything you want to track over time.

### Mention People
@mentions let you track relationships and interactions. You can see all entries mentioning a person from the People page.

### Experiment with Entry Types
Don't limit yourself to Journal entries. Try Dreams for goals, Highlights for special moments, and Ideas for creative thinking.

### Star What Matters
Use the star feature to mark entries you want to revisit. Starred entries are easy to filter and find later.

### Review the Insights
Periodically check the Insights page to see your writing patterns, streaks, and trends.
