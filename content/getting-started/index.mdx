## Quick Checklist

- [ ] Create your account (email/password or Google/GitHub)
- [ ] Verify your email (password accounts only)
- [ ] Write your first journal entry on the Write page
- [ ] Try a different entry type (Dreams, Highlights, Ideas, Wisdom, or Notes)
- [ ] Use `@mentions` and `#tags` to connect your thoughts
- [ ] Check the Insights page to see your first streak

## Prerequisites

| Requirement          | Details                                                |
| -------------------- | ------------------------------------------------------ |
| **Browser**          | Chrome, Firefox, Safari, or Edge (latest two versions) |
| **Email**            | A valid email address for account creation             |
| **OAuth (optional)** | A Google or GitHub account for one-click sign-in       |

No software installation is required. Reflecto runs entirely in the browser.

## What Makes Reflecto Different

**Six specialized entry types.** Journal, Dreams, Highlights, Ideas, Wisdom, and Notes each have tailored metadata -- mood tracking for journals, status progression for ideas, importance levels for highlights, and more.

**Connected thinking.** Every `@mention` and `#tag` you write is extracted automatically and linked across entries, building a web of relationships and topics over time.

**Insights and streaks.** Activity heatmaps, writing streaks, mood trends, and entry statistics help you understand your reflection patterns without extra effort.

**Rich editor.** TipTap-powered editor with markdown shortcuts, formatting toolbar, image uploads, and auto-save. Your work is continuously backed up as you type.

**Privacy first.** Secure authentication (email/password with bcrypt hashing, or OAuth via Google/GitHub), encrypted sessions, and private-by-default entries. Only you can see your reflections.

## Core Features at a Glance

| Feature           | Description                                                                        |
| ----------------- | ---------------------------------------------------------------------------------- |
| **Rich Editor**   | TipTap editor with markdown, bold/italic/underline, headings, lists, links, images |
| **Journal**       | Daily entries with categories, mood (1-5), and calendar navigation                 |
| **Dreams**        | Future aspirations and long-term visions                                           |
| **Highlights**    | Meaningful moments with Low/Medium/High importance                                 |
| **Ideas**         | Creative concepts with Seed/Draft/Refined/Implemented status                       |
| **Wisdom**        | Quotes, thoughts, facts, excerpts, and lessons with source tracking                |
| **Notes**         | Quick captures with color coding and pinning                                       |
| **Tags & People** | Auto-extracted `#tags` and `@mentions` with full management pages                  |
| **Insights**      | Activity heatmap, writing streaks, entry stats, and charts                         |
| **Customization** | Themes, font sizes, editor modes, feature toggles, and display settings            |
