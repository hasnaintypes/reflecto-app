# Navigation Guide

Reflecto uses a sidebar-based layout. The sidebar on the left controls navigation; the main content area on the right displays the current page.

## Sidebar Structure

The sidebar is divided into two sections separated by a divider.

### Primary Navigation

| Icon | Page | Purpose |
|---|---|---|
| Feather | **Write** | Create or edit entries. Your main workspace. |
| Book | **Journal** | Browse entries by date using a calendar view. |
| Rotate | **Reflect** | Timeline of recent entries across all types. |
| Chart | **Insights** | Activity heatmap, writing streaks, mood trends, and stats. |

### Categories

| Icon | Page | Purpose |
|---|---|---|
| Moon | **Dreams** | View and create dream/aspiration entries. |
| Heart | **Highlights** | View and create highlight entries with importance levels. |
| Hash | **Tags** | Browse all tags, see entry counts, manage tag groups. |
| At | **People** | Browse all @mentioned people, see mention counts, manage groups. |
| Pin | **Notes** | View and create quick notes with colors and pinning. |
| Gem | **Wisdom** | View and create wisdom entries (quotes, thoughts, lessons). |
| Lightbulb | **Ideas** | View and create idea entries with status tracking. |

### Profile

Your profile avatar sits at the bottom of the sidebar. Click it to access **Settings**, **Profile**, and **Sign Out**.

<Callout type="info">
Dreams and Notes can be toggled off in Settings > Preferences. When disabled, they disappear from the sidebar.
</Callout>

## Desktop vs Mobile

<Tabs items={['Desktop', 'Mobile & Tablet']}>
  <Tabs.Tab>
    **Desktop (1024px and above)**

    - Sidebar is expanded by default showing icons and labels
    - Click the collapse button (chevron icon at bottom) to shrink it to icons only
    - Enable **Auto-Hide Navigation** in Settings to make the sidebar fade away and reappear on hover
    - Collapsed/expanded state persists as you navigate between pages
  </Tabs.Tab>
  <Tabs.Tab>
    **Mobile and Tablet (below 1024px)**

    - Sidebar auto-collapses to icon-only mode
    - On smaller screens, tap the menu icon to open the sidebar as an overlay
    - Tap outside or navigate to a page to close it
    - The full content area is available without the sidebar taking space
  </Tabs.Tab>
</Tabs>

## Page-by-Page Overview

### Write

**URL:** `/write` or `/write?type=dream&date=2026-03-08`

The default landing page. Select an entry type from the dropdown, write in the editor, and auto-save handles the rest. Query parameters let you deep-link to a specific type or date.

### Journal

**URL:** `/journal`

A calendar-based view of your entries. Use the month navigation arrows to browse past months. Click a date to see all entries from that day. Click an entry to open it in the editor.

### Reflect

**URL:** `/reflect`

A chronological timeline of recent entries. Scroll down to load more. Filter by entry type. Click any entry card to edit it.

### Insights

**URL:** `/insights`

Your analytics dashboard. Includes an activity heatmap (contribution graph), current writing streak, total entry/tag/people counts, and charts breaking down entry types and categories. Click heatmap cells to view entries from that day.

### Workspace Pages

**URLs:** `/dreams`, `/highlights`, `/notes`, `/wisdom`, `/ideas`

Each workspace lists entries of that type with type-specific metadata displayed (importance for highlights, status for ideas, color for notes, etc.). Create new entries or click existing ones to edit.

### Tags

**URL:** `/tags`

All tags extracted from your entries, listed with entry counts. Tags can be organized into groups. Click a tag to view every entry that uses it.

### People

**URL:** `/people`

All @mentioned people with mention counts. People can be organized into groups (Work, Family, Friends). Click a person to see every entry that mentions them.

## Calendar Navigation

The Journal page features a monthly calendar view:

- **Month arrows** -- navigate forward and backward by month
- **Date cells** -- dates with entries are visually marked; click to filter
- **Calendar strip** -- a compact alternative showing the current week
- **Today indicator** -- today's date is highlighted

## Search and Filtering

Currently you can filter entries by:

- **Date** -- select from the Journal calendar
- **Entry type** -- navigate to the corresponding workspace page
- **Tags** -- click a tag on the Tags page
- **People** -- click a person on the People page

Full-text search across all entries is planned for a future release.

## URL Deep Linking

Bookmark specific views for quick access:

```
/write                        Write page (today's journal)
/write?type=idea              Write a new idea
/write?date=2026-03-01        Edit a specific date's entry
/journal                      Calendar view
/insights                     Analytics dashboard
/tags                         Tag management
/people                       People management
/settings                     App settings
```

## Settings and Profile

Access settings by clicking your profile avatar in the sidebar bottom-left corner, then selecting **Settings**. Key areas:

- **Appearance** -- theme, font size, editor mode
- **Preferences** -- default entry type, bulleted mode, auto-hide nav, special dates
- **Security** -- change password (email accounts only)
- **Profile** -- display name, profile image
