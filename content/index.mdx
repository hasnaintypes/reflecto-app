# Welcome to Reflecto

**Reflecto** is a private journaling application designed for mindful daily reflection. Built for people who want a beautiful, distraction-free space to capture thoughts, track patterns, and grow through writing.

## What is Reflecto?

Reflecto is a **reflection ecosystem**. Every entry you write becomes part of a connected web of thoughts, people, tags, and insights that help you understand yourself better over time.

## Core Features

| Feature           | Description                                                                                    |
| ----------------- | ---------------------------------------------------------------------------------------------- |
| **Rich Editor**   | TipTap-powered editor with markdown shortcuts, formatting, @mentions, #tags, and image uploads |
| **Journal**       | Daily entries with categories, mood tracking, and calendar navigation                          |
| **Dreams**        | Capture aspirations and long-term visions                                                      |
| **Highlights**    | Record meaningful moments with importance levels                                               |
| **Ideas**         | Track ideas from seed to draft with status progression                                         |
| **Wisdom**        | Collect quotes, thoughts, facts, excerpts, and life lessons                                    |
| **Notes**         | Quick captures with color coding and pinning                                                   |
| **Insights**      | Activity heatmaps, writing streaks, stats, and charts                                          |
| **Tags & People** | Auto-extracted from entries with full management and grouping                                  |
| **Customization** | Themes, editor preferences, feature toggles, and display settings                              |

## Design Philosophy

Reflecto is built on three principles:

1. **Privacy First** — Your reflections are personal. Secure authentication and data handling are built in from day one.
2. **Beautiful Minimalism** — A dark-first, premium aesthetic that makes writing feel special. Serif headings, smooth animations, and intentional whitespace.
3. **Connected Reflection** — Tags, people, and insights link your entries together, revealing patterns you would never notice in a traditional journal.
