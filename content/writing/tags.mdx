# Tags

Tags are the connective tissue of your Reflecto experience. They link entries together, reveal patterns, and help you navigate your thoughts across time.

<Callout type="info">
Tags in Reflecto are automatically extracted from your entries and tracked across all entry types. Use them to create your own personal knowledge graph.
</Callout>

---

## What Are Tags?

Tags are keywords prefixed with `#` that categorize and connect your entries.

### How Tags Work

1. **Write with #hashtags** — Type `#work`, `#fitness`, `#learning` in your entries
2. **Auto-extraction** — Tags are extracted when you save
3. **Suggestion dropdown** — Existing tags appear as you type
4. **Entry linking** — Tags create bidirectional links between entries
5. **Private & scoped** — Your tags are completely private to you

### Tag Syntax

```markdown
Had a productive meeting about #project-alpha today.
Discussed #design and #ux decisions with @Alice.
Feeling motivated about #learning and #growth.
```

When you save this entry, five tags are automatically extracted:
- `#project-alpha`
- `#design`
- `#ux`
- `#learning`
- `#growth`

## Creating Tags

### Inline in Editor

The easiest way to create tags is while writing:

<Steps>
### Type `#`

The suggestion dropdown appears instantly

### Start Typing

See existing tags that match what you type

### Select Existing or Create New

- **Arrow keys** — Navigate suggestions
- **Enter** — Select highlighted tag
- **Keep typing** — Create a new tag
- **Space/Tab** — Finalize tag

### Tag Is Created on Save

New tags are added to your tag library when you save the entry
</Steps>

### Tag Suggestions

As you type, the dropdown shows:
```
┌─────────────────────────────┐
│ #work (23 entries)          │
│ #workout (12 entries)       │
│ #working-out (3 entries)    │
│ ───────────────────────────  │
│ Create "#worklife"          │
└─────────────────────────────┘
```

- **Matching tags** — Filtered by what you've typed
- **Entry counts** — How many entries use each tag
- **Create new option** — Always available at the bottom

<Callout>
**Pro tip:** Check suggestions before creating new tags to maintain consistency and avoid duplicates.
</Callout>

## Tag Naming Best Practices

### Use Lowercase

Tags are case-insensitive but displayed as you first type them:
- ✅ `#project-alpha`
- ❌ `#Project-Alpha` or `#PROJECT-ALPHA`

### Use Hyphens for Multi-Word Tags

- ✅ `#product-launch`
- ❌ `#product launch` (space breaks the tag)
- ❌ `#productLaunch` (camelCase is less readable)

### Be Consistent

Avoid creating similar tags:
- ✅ Choose one: `#work`
- ❌ Don't use: `#work`, `#working`, `#workplace`, `#job`

### Create Hierarchies

Use prefixes for related tags:
- `#project-alpha`, `#project-beta`, `#project-gamma`
- `#book-fiction`, `#book-nonfiction`, `#book-reference`
- `#exercise-running`, `#exercise-yoga`, `#exercise-strength`

### Keep Tags Focused

- ✅ `#meditation` — Clear and specific
- ❌ `#trying-to-meditate-more` — Too verbose

### Use Descriptive Names

- ✅ `#quarterly-review`
- ❌ `#qr` (ambiguous)

<Callout type="warning">
Tag names cannot contain spaces, special characters (except hyphens), or emoji. Stick to letters, numbers, and hyphens.
</Callout>

## Tag Groups

Organize related tags into named groups for better clarity.

### Creating a Tag Group

<Steps>
### Select Tags

On the Tags page, click the checkbox next to multiple tags:
- `#project-alpha`
- `#project-beta`
- `#project-gamma`

### Click "Create Group"

Button appears at the top when tags are selected

### Name the Group

Enter a group name, e.g., "Projects"

### Group Is Created

Tags are visually grouped together on the Tags page
</Steps>

### Example Groups

**Work:**
- `#meetings`
- `#project-alpha`
- `#team`
- `#deadlines`

**Health:**
- `#exercise`
- `#nutrition`
- `#meditation`
- `#sleep`

**Learning:**
- `#books`
- `#courses`
- `#articles`
- `#podcasts`

**Personal:**
- `#family`
- `#friends`
- `#hobbies`
- `#travel`

### Managing Groups

- **Rename** — Click the group header to edit name
- **Reorder** — Drag groups to rearrange (coming soon)
- **Add tags** — Drag tags into existing groups
- **Remove tags** — Uncheck tag and remove from group
- **Delete group** — Click the trash icon (tags remain)

<Callout type="info">
Groups are purely organizational — they don't affect how tags work in entries, they just make the Tags page easier to navigate.
</Callout>

## Editing Tags

### Renaming a Tag

<Steps>
### Navigate to Tags Page

Click **Tags** in the sidebar

### Click the Tag

Select the tag you want to rename

### Click Edit Button

Found in the tag detail view header

### Enter New Name

Type the new tag name

### Confirm

Changes apply to all entries using this tag
</Steps>

<Callout type="warning">
Renaming a tag updates it across **all entries**. This is useful for fixing typos or consolidating similar tags.
</Callout>

### Merging Tags

To merge duplicate tags:

1. Rename one tag to match another
2. System detects duplicate
3. Confirms merge
4. All entries now use the single tag

**Example:**
- You have `#work` (40 entries) and `#working` (5 entries)
- Rename `#working` to `#work`
- Now `#work` has 45 entries
- `#working` no longer exists

## Deleting Tags

### When to Delete Tags

- **Typos** — Created by mistake (better to rename)
- **Irrelevant** — Tag no longer useful
- **Cleanup** — Consolidating tags

### How to Delete

<Steps>
### Navigate to Tags Page

Click **Tags** in the sidebar

### Click the Tag

Open the tag detail view

### Click Delete Button

Usually a trash icon

### Confirm Deletion

Warning appears: "This tag will be removed from all entries"

### Tag Is Deleted

Removed from your library and all entries
</Steps>

<Callout type="warning">
Deleting a tag is **permanent** and removes it from all entries. Consider renaming instead if you might want the data later.
</Callout>

## Tag Analytics & Insights

View tag statistics and patterns in the **Insights** page.

### Tag Insights Include:

**Usage Frequency:**
- Total unique tags
- Most used tags
- Least used tags
- Tags per entry average

**Trends:**
- Tag usage over time (line chart)
- New tags created by month
- Trending tags (increasing usage)
- Declining tags (decreasing usage)

**Relationships:**
- Tags frequently used together
- Entry type distribution per tag
- Mood correlation with tags (e.g., `#exercise` correlates with high mood)

**Heatmap:**
- Tag activity heatmap showing when tags are used most
- Day-of-week patterns (e.g., `#work` peaks Mon-Fri)

<Callout>
Insights help you understand your focus areas, spot trends, and identify tags worth organizing into groups.
</Callout>

## Best Practices

### 1. Start Simple

Begin with 5-10 core tags:
- `#work`, `#personal`, `#health`, `#learning`, `#goals`

Expand as needed.

### 2. Review Tags Monthly

Check the Tags page monthly:
- Merge similar tags
- Delete unused tags
- Create groups for organization
- Identify trending topics

### 3. Use Tags Consistently

Decide on a naming convention early:
- All lowercase
- Hyphens for multi-word
- Hierarchical prefixes

### 4. Don't Over-Tag

3-5 tags per entry is usually enough. Too many tags dilute their value.

### 5. Combine with People

Tags + @mentions create rich context:
- "Meeting with @Alice about #project-alpha and #design"

### 6. Check Insights

Review tag analytics to understand your focus areas and spot patterns.

### 7. Use Groups

Once you have 20+ tags, organize them into groups for easier navigation.

### 8. Tag Retroactively

Edit old entries to add tags you weren't using before. This builds richer connections.

## Troubleshooting

### Tag not appearing in suggestions?

- Type at least 2 characters after `#`
- Check that you've created the tag before (in another entry)
- Verify entry was saved successfully

### Tag not extracted from entry?

- Ensure tag has `#` prefix
- Check there's no space after `#` (e.g., `# work` won't work)
- Save the entry (tags extract on save, not during typing)

### Can't delete a tag?

- Ensure you have permission (shouldn't be an issue for your own tags)
- Check browser console for errors
- Refresh the page and try again

### Tag count seems wrong?

- Counts update when entries are saved
- Refresh the Tags page
- Check that entries weren't deleted

### Groups not saving?

- Ensure you've selected multiple tags
- Enter a group name
- Check internet connection
